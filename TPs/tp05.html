TP N°5 de compression

Le but de ce TP est d'effectuer de calculer la performance d'un méthode de compression par modélisation de contexte, et d'effectuer une synthèse de l'ensemble des méthodes de compression

Exercice 1: Compression par modélisation de contexte

On voudrait tester l'efficacité d'un codage de type PPM (sans écrire la méthode de compression/décompression elle-même).
  1. Pour réaliser cela, écrire une fonction qui calcule les probabilités conditionnelles d'ordre n (=pour les n caractères précédents) sur une source. Cela revient, pour tous les blocs de taille n, à calculer l'histogramme du caractère qui suit ce bloc (exemple: pour (..., aaba: a=14, b=7, ...), cela signifie que dans la source considérée, le bloc aaba est suivi dans 14 cas d'un a, et dans 7 cas d'un b). L'histogramme ainsi estimé est celui de N(Xp-n...Xp-1,Xp).
  2. Ecrire une fonction qui à partir du contexte (= la valeur du bloc), calcule la probabilité du caractère suivant (par exemple, si ce caractère est b, dans le contexte aaba, alors la probabilité de b dans ce contexte est 7/21). Puis, retourne la self -information associée (dans ce même exemple: - log(7/21)/log(2)), qui correspond au nombre de bits optimal pour coder un symbole ayant cette probabilité dans ce contexte (qui est atteinte pour un codage arithmétique).

    On remarquera que la probabilité conditionnelle dans le contexte est estimée comme N(Xp-n...Xp-1,Xp) / Somme( N(Xp-n...Xp-1,Xp) pour tous les Xp existants dans ce contexte).

  3. Calculer la somme des self-informations de chaque caractère sur la totalité de la source. On utilisera cette quantité comme estimation de la compression PPM.
  4. Calculer l'entropie pour la loi jointe (loi jointe = histogrammes locaux renormalisé par N-(n+1) qui est le nombre de contextes d'ordre de n rencontré dans une source de longueur N (taille du fichiers)).

    La loi jointe est estimée comme Pr[Xp-n...Xp-1,Xp]=N(Xp-n...Xp-1,Xp) / Somme( N(Xp-n...Xp-1,Xp) pour tous les Xp et tous les contextes (Xp-n...Xp-1).

  5. Calculer l'entropie H(Xp) et l'entropie conditionnelle H(Xp|(Xp-n...Xp-1). On rappelle que H(Xp) est l'entropie de la loi du symbole.
